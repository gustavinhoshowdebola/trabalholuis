Métodos

enqueue(elemento) - adiciona elemento no final da fila

dequeue() - remove o primeiro elemento da fila

peek() - mostra o primeiro elemento sem remover

is_empty() - verifica se a fila está vazia

is_full() - verifica se a fila está cheia

size() - retorna o tamanho atual da fila

clear() - limpa a fila
